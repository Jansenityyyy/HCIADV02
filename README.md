# 🏫 Amore Academy — School Website

> *"Shaping Minds, Building the Future"*

A multi-page fictional school website for **Amore Academy**, a K-12 institution based in Metro Manila, Philippines. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## 📄 Pages

| File | Description |
|---|---|
| `Home.HTML` | Landing page — hero, quick links, programs preview, news |
| `about.html` | School history, mission & vision, core values |
| `academics.html` | JHS & SHS programs, tracks & strands |
| `admissions.html` | Enrollment requirements and procedure |
| `apply.html` | Online application form |
| `news.html` | News & announcements |
| `legal.html` | Privacy Policy, Terms of Use, Accessibility |

---

## 🎨 Stylesheets

The CSS is split into a global base file and per-page files:

```
main.css          ← design tokens, reset, navbar, footer, shared components
home.css          ← hero, quick links, programs, news preview
about.css         ← page hero, subnav, timeline, mission/vision, values grid
academics.css     ← tracks, strands, program cards
admissions.css    ← requirements, enrollment steps
apply.css         ← application form
news.css          ← news grid, article cards
legal.css         ← legal page layout
```

## 🏫 About the School (Fictional)

**Amore Academy** was founded in the early 1970s by a group of dedicated educators in Metro Manila. Starting with fewer than 200 students, it has grown into a community of over 5,000 learners offering complete Junior High School and Senior High School programs under the Philippine K-12 curriculum.

### Mission
Provide holistic, values-based education that empowers learners to achieve academic excellence, develop strong moral character, and become responsible members of society.

### Vision
A premier institution in the Philippines producing globally competitive graduates who are deeply rooted in Filipino values and equipped to lead with integrity and purpose.

### Core Values
- **Service** — Compassion in action
- **Honor** — Dignity and respect for all
- **Integrity** — Honesty in academics and life
- **Excellence** — Giving one's best always
- **Loyalty** — Steadfast commitment to the community

## 🛠️ Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties (design tokens), BEM-inspired class naming, responsive grid/flexbox
- **Vanilla JavaScript** — SPA-style tab switching, sticky subnav, animations
- **No frameworks** — zero dependencies
